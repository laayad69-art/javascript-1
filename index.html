دروس JavaScript

الدرس الأول

السؤال الأول

اكتب كود JavaScript الذي يعرض "Reward: steak" إذا كانت قيمة المتغير science تساوي 100.

solution-1.js 
let science = 100;
if (science == 100) {
  console.log("Reward: steak");
}
السؤال الثاني

اكتب كود JavaScript الذي يعرض "Reward: steak" إذا كانت science تساوي 100، أو "Reward: ice cream" إذا كانت 80 أو أكثر، أو "No games for 1 week." في الحالات الأخرى.

solution-2.js 
let science = 80;
if (science == 100) {
  console.log("Reward: steak");
} else if ( science >= 80 ){
  console.log("Reward: ice cream");
} else {
  console.log("No games for 1 week.");
}

السؤال الثالث

اكتب الشرط الذي يتحقق مما إذا كان numberOfPeople أكبر من أو يساوي 3 AND total يساوي 3333.

condition-3.js 
numberOfPeople >= 3 && total == 3333
السؤال الرابع

اكتب الشرط الذي يتحقق مما إذا كان numberOfPeople أكبر من أو يساوي 10 OR total أكبر من أو يساوي 12000.

condition-4.js 
numberOfPeople >= 10 || total >= 12000

الدرس الرابع

السؤال الأول

اكتب كود JavaScript الذي يتحقق مما إذا كانت النتيجة (score) أكبر من أو تساوي 80 وفي هذه الحالة يعرض "Test Passed"، وإذا كانت الرتبة (rank) تساوي 1 يعرض أيضًا "Trophy Acquired".

solution-4-1.js 
let score = 90;
let rank = 1;
if (score >= 80) {
  console.log("Test Passed");
  if (rank == 1) {
    console.log("Trophy Acquired");
  }
}

إجابات المراجعة

اكتب الشرط الذي يتحقق مما إذا كانت النتيجة (score) تساوي 50:
score == 50
اكتب الجزء المكمل للكود التالي: else if (____):
else if (A < B)
اكتب الشرط الذي يتحقق مما إذا كانت A أقل من B:
A < B
اكتب الشرط الذي يتحقق مما إذا كانت النتيجة (score) أكبر من أو تساوي 50:
score >= 50
ما هو الرمز المستخدم للعامل المنطقي "و" (AND) في JavaScript؟
&&
ما هو الرمز المستخدم للعامل المنطقي "أو" (OR) في JavaScript؟
||
اكتب الشرط الذي يتحقق مما إذا كان numberOfPeople أقل من أو يساوي 10 OR weight أقل من أو يساوي 800:
numberOfPeople <= 10 || weight <= 800
ماذا يحدث إذا كتبنا if (numberOfPeople = 10)؟
ينتج عنه خطأ
اكتب كود JavaScript الذي يتحقق مما إذا كانت math تساوي 100 وفي هذه الحالة يعرض "Math is 100 points"، وإذا كانت science تساوي 100 يعرض أيضًا "Math and science are 100 points".
let math = 100; let science = 100; if (math == 100) { console.log("Math is 100 points"); if (science == 100) { console.log("Math and science are 100 points"); } }
